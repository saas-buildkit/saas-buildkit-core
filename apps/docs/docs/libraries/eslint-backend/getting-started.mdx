# Getting Started

---

:::note
This is a package ESLint configuration for the Softkit ecosystem. You can use it if you want, but please be aware that we will update it based on our needs.
:::

## Installation

To get started, install the library using the following command:

```bash
yarn add -D @softkit/eslint-config
```

## Next Steps

After installation, you can extend this ESLint configuration to maintain code quality and consistency across your projects. Proceed to the [features documentation](/libraries/eslint-backend/features) to understand the default rules and plugins included.
