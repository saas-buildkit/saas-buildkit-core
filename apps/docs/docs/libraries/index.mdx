# Introduction to Softkit Libraries

Welcome to the documentation for Softkit's libraries. You'll find detailed instructions to aid you in utilizing our tools effectively. If you require further assistance, our support resources are here to help.

## Available Libraries

Softkit offers a wide range of libraries designed to streamline and enhance your development process. Each library is purpose-built to address specific aspects of software development, ensuring you have the right tools at your fingertips.

- **[Async Storage](/libraries/async-storage/getting-started)**: Efficient asynchronous data storage solutions.
- **[Auth](/libraries/auth/getting-started)**: Robust authentication and authorization mechanisms.
- **[Bootstrap](/libraries/bootstrap/getting-started)**: Quick start tools for application setup.
- **[Clients](/libraries/clients/getting-started)**: Simple HTTP Client for the platform application.
- **[Common Types](/libraries/common-types/getting-started)**: Shared type definitions across services.
- **[Config](/libraries/config/getting-started)**: Configuration management made simple.
- **[Crypto](/libraries/crypto/getting-started)**: Cryptographic functions for secure data handling.
- **[ESLint Backend](/libraries/eslint-backend/getting-started)**: Linting rules for backend consistency.
- **[Exceptions](/libraries/exceptions/getting-started)**: Standardized exception handling following [RFC7807](https://datatracker.ietf.org/doc/html/rfc7807).
- **[File Storage](/libraries/file-storage/getting-started)**: Solutions for file management and storage.
- **[Health Check](/libraries/health-check/getting-started)**: Monitor and ensure the operational health of your services running on AWS.
- **[I18n](/libraries/i18n/getting-started)**: Internationalization support for global applications.
- **[Logger](/libraries/logger/getting-started)**: Logging utilities for monitoring and debugging.
- **[Mail](/libraries/mail/getting-started)**: Email integration and management tools.
- **[Redis](/libraries/redis/getting-started)**: Redis utilities for fast data caching and retrieval.
- **[Resource Plugin](/libraries/resource-plugin/getting-started)**: Generators for convenient development of new resources and their CRUD operations.
- **[Server HTTP Client](/libraries/server-http-client/getting-started)**: HTTP client for server-side requests.
- **[String Utils](/libraries/string-utils/getting-started)**: Utility functions for string manipulation.
- **[Swagger Utils](/libraries/swagger-utils/getting-started)**: Tools for working with Swagger and OpenAPI.
- **[Test Utils](/libraries/test-utils/getting-started)**: Testing utilities to ensure your code meets quality standards.
- **[TypeORM](/libraries/typeorm/getting-started)**: Integrations and utilities for working with TypeORM.
- **[TypeORM Service](/libraries/typeorm-service/getting-started)**: Service layer abstraction for TypeORM.
- **[Validation](/libraries/validation/getting-started)**: Data validation tools for robust error checking.

## Getting Started

To get started with the Softkit Libraries, simply pick the library that fits your needs and dive into the respective documentation. Each library's documentation includes setup instructions, usage examples, and configuration options to help you integrate it into your projects seamlessly.
