# Default Exclusion Lists

---

## Overview

The library includes default exclusion lists for CREATE, UPDATE, and select Data Transfer Objects (DTOs). These lists are integral to the data handling process, ensuring that certain entity properties are not exposed or modified unintentionally during various operations.

### Features

- [**Default Entity Exclude List**](/api/typeorm/src/variables/DEFAULT_ENTITY_EXCLUDE_LIST/): Specifies common properties to be excluded across various entity operations.
- [**Default Create Entity Exclude List**](/api/typeorm/src/variables/DEFAULT_CREATE_ENTITY_EXCLUDE_LIST/): Extends the default entity exclude list for CREATE operations
- [**Default Update Entity Exclude List**](/api/typeorm/src/variables/DEFAULT_UPDATE_ENTITY_EXCLUDE_LIST/): Similar to the create list, but tailored for UPDATE operations.
